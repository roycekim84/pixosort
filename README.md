# PixoSort

**PixoSort: Pixel Art Puzzle**  
**픽소소트: 도트 컬러 퍼즐**

PixoSort is a Flutter mobile puzzle game where players sort mixed color blocks to restore cute pixel art images.

픽소소트는 섞여 있는 색상 블록을 같은 색끼리 정리하면 숨겨진 도트 그림이 완성되는 모바일 퍼즐 게임입니다.

## Core Loop

1. Choose a stage.
2. Sort mixed color blocks into matching tubes.
3. Completed colors fill the pixel art board.
4. Restore the full image.
5. Save it to the collection book.

## Main Features

- 10 difficulty levels
- 100 built-in stages
- Pixel art restoration reward
- Photo-to-puzzle bonus mode
- Local collection book
- Offline-first design
- Android and iOS release target
