# PixoSort Concept Plan

## Product Name

- English: PixoSort: Pixel Art Puzzle
- Korean: 픽소소트: 도트 컬러 퍼즐

## One-Line Concept

Sort colors to restore pixel art.

색을 정리하면 도트 그림이 완성되는 컬러 소트 퍼즐 게임.

## Game Overview

PixoSort is a casual mobile puzzle game that combines tube-based color sorting with pixel art restoration.

Players move stacked color blocks between tubes. When all blocks of one color are gathered correctly, that color is applied to the hidden pixel art board. The stage is cleared when every color has been sorted and the full image is restored.

The game provides built-in stages and a bonus mode where users can turn their own photos into pixel color puzzles.

## Core Fun

- Simple color sorting rules
- Satisfying pixel art restoration
- Cute image collection
- Short one-hand mobile sessions
- Increasing difficulty through image size, color count, and tube count
- Personalized replay value through photo puzzle generation

## Target Users

- Casual puzzle players
- Color sort and water sort game fans
- Pixel art and cute illustration fans
- Users who enjoy collecting completed images
- Users who want to turn personal photos into simple games

## Key Differentiation

Most color sort games end when the colors are organized. PixoSort adds a visible creative reward: each completed color restores part of a pixel art image.

The game is not only about solving a sorting puzzle. It is also about revealing and collecting cute pixel art.

## Main Modes

### Built-in Puzzle Mode

- 10 difficulty levels
- 10 stages per difficulty
- 100 built-in stages total
- Sequential or difficulty-based unlock structure
- Completed images are saved into the collection book

### Photo Puzzle Mode

- User selects a photo
- App crops it to a square
- App converts it to pixel art
- App extracts a limited color palette
- App generates a playable tube sort puzzle
- User restores the generated image through gameplay

## Difficulty Structure

Difficulty is controlled by:

- Pixel art resolution
- Color count
- Tube count
- Empty tube count
- Color similarity
- Shuffle complexity
- Whether the target preview is fully visible

Initial release should mainly use resolution, color count, and tube count.

## Theme Direction

The built-in stages should share a warm and cute pixel-art world.

Suggested themes:

- Food
- Animals
- Cats
- Cat jobs
- Harbors
- Markets
- Travel scenes
- Fantasy locations
- Legendary fish and treasures

## Main Screens

- Home
- Difficulty Select
- Stage Select
- Puzzle Play
- Result
- Collection
- Photo Puzzle Creator
- Settings

## Monetization Direction

PixoSort should be playable offline and enjoyable without aggressive monetization.

Possible monetization:

- Rewarded ads for hints
- Rewarded ads for extra undo
- Rewarded ads for additional photo puzzle generation
- Remove ads purchase
- Premium theme packs
- Unlimited photo puzzle mode
- High-resolution export

## Initial Release Goal

The first release should be a complete small mobile puzzle game for Android and iOS.

Target release contents:

- 100 built-in stages
- Photo puzzle mode
- Local collection book
- Save/share completed images
- Local progress storage
- Optional ads and minimal in-app purchases

## Development Principle

- Offline-first
- No required backend for initial release
- Simple and clear rules
- Short play sessions
- Strong visual reward
- Cute, readable pixel art
- Mobile-friendly touch controls
